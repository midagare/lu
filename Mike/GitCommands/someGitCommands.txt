Para subir cambios

1. Agregar
git add filename

o agregar todos:
git add .

2. commit
git commit -m "message"

3. push
git push origin branchName
e.g.
git push origin master



Para bajar cambios
git pull origin master


Para ver cambios
git status

Para ver commits
git log
git log -2

Cambiar de rama
git checkout branchName

Crear rama
git checkout -b newBranchName
